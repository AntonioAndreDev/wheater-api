Descrição
O Weather App é uma aplicação web que permite aos usuários pesquisar informações sobre o clima de qualquer cidade do mundo. Ele foi projetado para ser responsivo e acessível, garantindo uma experiência perfeita em todos os dispositivos e tamanhos de tela.

Recursos
- Pesquisar informações sobre o clima pelo nome da cidade
- Exibir condições climáticas atuais, incluindo temperatura, umidade, velocidade do vento e descrição do tempo

Tecnologias Utilizadas
O Weather App utiliza as seguintes tecnologias e ferramentas:
- HTML5, CSS3 e JavaScript para o desenvolvimento front-end
- npm (Node Package Manager) para gerenciamento de pacotes e instalação de dependências
- Webpack para empacotamento e minificação de arquivos
- Babel para transpilar o código JavaScript para melhor compatibilidade com os navegadores

Uso
- Abra o Weather App em seu navegador da web.
- Insira o nome da cidade desejada na barra de pesquisa.
- Pressione a lupa "Pesquisar" ou pressione Enter.
- As informações climáticas atuais serão mostradas na tela.
